# Option Pricing Using Machine Learning

This repository implements a neural network-based approach for pricing financial options and computing implied volatilities. The solution leverages Artificial Neural Networks (ANNs) to significantly accelerate computations based on models like Black-Scholes and Heston stochastic volatility.

## Features

- **Fast Option Pricing:** Utilizes ANNs to speed up option price computations.
- **Efficient Implied Volatility Calculation:** Computes implied volatilities quickly using trained models.
- **Supports Key Financial Models:** Includes implementations for Black-Scholes and Heston models.
- **GPU Optimization:** Offers substantial speed improvements, especially when using GPU acceleration.
